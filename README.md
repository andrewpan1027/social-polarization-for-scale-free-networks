# social-polarization-for-scale-free-networks

This repository contains the source code for the paper **"Continuous Growth of Social Polarization for Scale-Free Networks"**, currently under review in the *Chaos* journal.

This project is inspired by and builds upon the excellent work at:

**[Connectivity-Polarization Transition](https://github.com/complexity-science-hub/ConnectivityPolarizationTransition)**

Special thanks to:
- **Thurner, S., Hofer, M., & Korbel, J. (2025).** "Why more social interactions lead to more polarization in societies." *Proceedings of the National Academy of Sciences*, 122(44), e2517530122. https://doi.org/10.1073/pnas.2517530122
